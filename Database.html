Backend: Databases

Efficiently managing and organizing large amounts of data with databases

Databases Used During This Project

The purpose of databases is to provide a structured and organized way to store and manage data. Databases allow for efficient retrieval, modification, and querying of large amounts of data. They also provide mechanisms for ensuring data integrity and security, and enable multiple users or applications to access and modify data simultaneously. Overall, databases are a critical component of many software applications and are essential for managing and analyzing complex data.

phpMyAdmin

phpMyAdmin is a free and open-source web-based administration tool for managing MySQL and MariaDB databases. It allows users to manage database tables, execute SQL queries, import and export data, and perform other database management tasks through a graphical user interface.

MySQL Workbench

MySQL Workbench is a visual tool that provides a graphical interface for managing and working with MySQL databases. It allows users to design, model, create, and manage databases, tables, and relationships between them, as well as execute SQL queries and scripts.

ClearDB

ClearDB is a cloud-based database service that provides MySQL databases as a service on various cloud platforms, including Heroku.

Process of Choosing a Database

Initially, I utilized phpMyAdmin as it was a convenient method to create and link the database to the code. However, I adapted to the class curriculum and switched to using MySQL Workbench by connecting the phpMyAdmin database to the Workbench. Ultimately, to host the banking platform on Heroku, I opted for ClearDB as it was a compatible database with Heroku.
