Building Materials

Node JS

Node.js is a server-side JavaScript runtime environment that enables developers to build fast and scalable network applications using an event-driven, non-blocking I/O model.

Sequelize

Sequelize is a Node.js ORM tool that simplifies working with relational databases by mapping database tables to JavaScript objects. It supports multiple database management systems and advanced features such as associations and transactions.

Express JS

Express is a flexible and lightweight Node.js web application framework used for building APIs, web, and mobile applications. It simplifies HTTP request handling, routing, and middleware management, and has a rich ecosystem of third-party modules.

MySQL

MySQL is an open-source relational database management system used for storing and managing structured data, widely used in web applications and content management systems due to its scalability, performance, and advanced features.
