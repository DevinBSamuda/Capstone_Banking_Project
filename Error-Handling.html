Error Handling

Error Handling and Flash Messages

Error handling code is used to anticipate and manage potential errors in a software application, preventing crashes and improving user experience. Flash messages provide immediate feedback to the user, reducing frustration and enhancing the overall user experience.
By implementing error handling techniques and providing informative flash messages, our application can provide reliable and responsive services, enhancing the overall user experience.

Flash and Error Messages Used Within the Application

When incorrect information, such as non-existent user credentials, is entered, an error message will appear. This will be indicated by a message below the navigation bar which reads "Invalid email or password".

When depositing or withdrawing money from the balance, a flash message will be displayed to inform the user whether the transaction was successful or not. The flash message will indicate whether the user has deposited or withdrawn a certain amount of money.

In case a user is unable to access the banking platform despite having valid credentials, an error message will be displayed as "An error has occurred". This message will also be used in other situations when other error options are not applicable.

If characters other than numbers are inputted or the amount exceeds a certain limit, an error will occur and the balance will not be updated.

If a user tries to sign up using an email that already exists in the database, an error flash message will appear, indicating that the email already exists. Moreover, if the password and confirm password do not match, an error message will be displayed, notifying the user that the passwords do not match.

When logging out, the user will be redirected back to their account. However, any attempt to make changes will result in an intentional error message being displayed, indicating that it is not possible to alter the account after logging out. This serves as a reminder to the user that any changes should be made before logging out.

The flash package in Node.js is used to display temporary messages to the user. These messages provide feedback about actions or errors and are stored in a session, making them accessible across multiple requests and responses.

Flash Package From Node

Using the flash package provided by Node after installation, flash message will appear if written correctly and used an if statement in the HTML file.
